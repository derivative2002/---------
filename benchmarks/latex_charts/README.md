# LaTeX风格图表

本目录包含使用matplotlib生成的LaTeX风格图表。

## 图表列表

1. **ranking_bar_chart** - 不同场景下的单位排名对比
2. **score_ladder** - 综合评分天梯榜
3. **radar_chart** - 多维度性能雷达图
4. **heatmap** - 场景排名热力图

## 文件格式

每个图表都提供两种格式：
- `.png` - 高分辨率位图（300 DPI）
- `.pdf` - 矢量图格式

## 使用说明

这些图表采用学术论文风格设计，可直接用于：
- 学术论文
- 技术报告
- 演示文稿

图表使用了Times New Roman（英文）和系统中文字体。
