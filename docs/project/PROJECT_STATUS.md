## 项目概览

本项目旨在建立一个基于兰彻斯特方程的数学模型，用于客观评估《星际争霸II》合作任务模式中的单位战斗效能。经过多次迭代和精细化调整，我们已经构建了一个理论严谨、实战验证、代码规范的评估框架。v2.5版本引入了人口税和指挥官经济模型，使评估结果更加准确。
