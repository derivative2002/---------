## 命名规范

- 主版本号：v1.0, v2.0 等
- 次版本号：v1.1, v1.2 等（小修订）
- 草稿版本：v1.0-draft1, v1.0-draft2 等

## 文件说明

每个版本目录应包含：
- `report.tex` - LaTeX源文件
- `report.pdf` - 编译后的PDF文件
- 其他必要的图片、数据文件等

## 编译说明

使用XeLaTeX编译：
```bash
cd 到对应版本目录
xelatex report.tex
```

或在主paper目录使用Makefile：
```bash
cd ../../
make
```
